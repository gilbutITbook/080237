TensorFlow.js 예제: 간단한 객체 탐지

삼각형과 직사각형을 분류하고 위치를 감지하는 모델을 훈련합니다.

설명

이 페이지는 간단한 합성 장면에서 타깃 도형을 분류하고 위치를 감지하는 사전 훈련된 객체 탐지 모델의 추론을 수행합니다. 모델은 tfjs-node로 훈련했습니다(train.js 참조).

감지하려는 타깃 객체는 삼각형과 직사각형입니다. 배경에는 모델이 무시해야 할 선분과 원이 등장합니다. 각 합성 장면은 하나의 타깃 객체만 포함합니다. 모델이 추론을 통해 생성한 (파란 상자로 표시된) 타깃 객체의 바운딩 박스를 (빨간 상자로 표시된) 정답 바운딩 박스와 비교할 수 있습니다. 또한 추론의 출력에는 도형의 클래스(삼각형 또는 직사각형)가 포함되어 있습니다.
